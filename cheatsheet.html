SLASHED Cheatsheet v0.4.6.0

Buildless CSS framework · 201 tokens · 647 utility classes · 50+ components · README · Docs

Philosophy: UI components are .cs-* prefixed. Layout primitives and utilities are unprefixed for ergonomic authoring. Framework uses cascade layers — any unlayered BEM class you write beats framework rules automatically.

Responsive Variants sm: 30em · md: 48em · lg: 64em · xl: 80em

Prefix any layout utility with a breakpoint to apply at that viewport and up. Spacing/typography is auto-fluid via clamp() tokens — responsive variants cover discrete layout switches and specific overrides.
.sm: / .md: / .lg: / .xl:Breakpoint prefixes (480 / 768 / 1024 / 1280 px)
.md:flex / .md:grid / .md:hiddenDisplay switches
.md:flex-row / .md:flex-colDirection switches
.md:grid-2 / -3 / -4Grid columns at breakpoint
.md:items-* / .md:justify-*Alignment at breakpoint
.md:gap-m / -l / -xlGap at breakpoint
.md:text-left / -center / -rightAlign at breakpoint
.md:p-0 / .md:pt-0 / .md:mt-0 / .md:gap-0Zero resets at breakpoint
.md:col-span-2 / -fullGrid span at breakpoint
.md:order-first / -last / -noneOrder at breakpoint
<!-- Stack on mobile, 3-col md+, 4-col lg+ -->
<div class="grid md:grid-3 lg:grid-4 gap-m md:gap-l">...</div>

<!-- Hidden below md, flex from md+ -->
<nav class="hidden md:flex items-center gap-l">...</nav>
