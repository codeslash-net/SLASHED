Design Tokens tokens-default.css · docs/TOKENS.md

Three-tier hierarchy. Primitives = raw scale values (--primary-100, --space-m). Semantic aliases = role-based names that resolve through primitives (--color-bg, --color-text) — components and utilities reference aliases. Instance tokens = component-scoped overrides (--btn-bg, --card-padding) for one-off customisation without specificity warfare.
